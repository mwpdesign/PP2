üîÑ Sequential Processing Workflow

When "Process Selected" is clicked:

  1. Shows confirmation dialog: "Process X orders?"
  2. If confirmed, stores remaining order IDs in sessionStorage
  3. Navigates to first order's process page: /logistics/orders/[id]/process
  4. Remaining orders can be processed sequentially from sessionStorage
sessionStorage.setItem('remainingOrderIds', JSON.stringify(['2', '3', '6'])); window.location.href = '/logistics/orders/1/process';
